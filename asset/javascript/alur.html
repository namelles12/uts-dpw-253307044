Beranda Alur Pendaftaran Registrasi Akun

4 Tahap Utama Pendaftaran

Berikut adalah prosedur ringkas yang harus dilalui oleh semua calon siswa baru:

  1. 1. Registrasi Akun

    Calon pendaftar membuat akun pendaftaran baru. Hal ini wajib agar data terekam secara unik di tabel pendaftaran mandiri sistem.

  2. 2. Mengisi Formulir

    Mengisi borang/form pendaftaran. Pastikan isian Nama dan 16 angka NIK diketik tanpa adanya modifikasi/rumpang dari dokumen asli kependudukan.

  3. 3. Upload File Bukti Fisik

    Lampirkan dokumen unggahan yang dipersyaratkan secara teknis. Jika jalur prestasi, harap upload piagam lomba/kejuaraan relevan.

  4. 4. Proses Verifikasi Admin

    Tahap akhir menanti konfirmasi, jika datamu terkonfirmasi solid dan tak ada berkas cacat, statusmu akan diverifikasi.
